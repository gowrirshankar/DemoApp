
dotnet -h --> To open the dotnet command help
git init --> initialize the git 
dotnet new gitignore --> Create git ignore file to add the ignored file from applications to this file
dotnet new globaljson --> Create new json file globally
git status --> to check the status of the git like on which branch we are and how many files are yet to commit
dotnet watch run -> to run the application
dotnet ef database -h --> to see the command for ef related
dotnet ef migrations add InitialCreate - o Data/Migrations --> to add the initial data migrations using ef in specific folder(Data/Migrations)
dotnet ef database update --> to update the database


node -v --> Node version
npm -v --> Npm version
ng version --> Angular CLI version
